README


To successfully run this game, make sure that &&&&&.mp3 is in the ‘music’ folder, 
ufodod.js is in the ‘js’ folder, background.png, wizardWalking.png, and ghostWalking.png are in the ‘images’ folder, and index.html is in the same directory as the folders. 


Move with the arrow keys, 
don’t let the ghost touch you to win.
